FIELD-DEBUG LESSON (for future Claude sessions)
═══════════════════════════════════════════════════════════════════════════

Claude Code's bash session had ELECTRON_RUN_AS_NODE=1 set in its env.
Consequence: any Electron .exe launched from that bash silently ran
as plain Node (because that env var makes Electron binaries skip chrome
and run as node). main.js crashed on `require('electron').app` → undefined.

Symptom: exe exits 0 immediately, no stdout, no log file, no process.
Double-clicking from Explorer works because Explorer doesn't have the
env var.

Fix: `env -u ELECTRON_RUN_AS_NODE /path/to/app.exe`

Memory updated to flag this for future sessions.
