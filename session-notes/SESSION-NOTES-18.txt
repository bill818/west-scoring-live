1. SERVER-SIDE COMPUTATION ENGINE — SHIPPED
   Moved all results computation from client to Worker. Previously pages
   received raw .cls data and did their own parsing; now Worker pre-computes
   everything and pages are thin render templates.

   Architecture:
   - Watcher POSTs parsed .cls data to Worker via /postClassData
   - Worker runs computeClassResults() — dispatches to computeHunterResults()
     or computeJumperResults() based on class type
   - Pre-computed results stored in KV: results:slug:ring:classNum
   - /getResults serves pre-computed object directly — cls_raw never sent
     to client
   - On CLASS_COMPLETE: frozen into D1 classes.final_results
   - /getResults priority: KV pre-computed → D1 final_results →
     computed-fallback → D1 raw entries

   Covers:
   - Hunter derby: per-judge phase cards, rankings, judge card totals,
     movement arrows, split decision detection
   - Hunter non-derby scored: per-judge scores, sequential layout, up to
     7 judges
   - Hunter forced/flat: place-only rendering
   - Jumper all 16 scoring methods: round data + per-round stats
   - Table III / Faults Converted: r1FinalTime = clock + jumpFaults +
     penaltySeconds
   - isFaultsConverted flag included in computed results

2. MULTI-ROUND NON-DERBY HUNTER — SHIPPED
   Non-derby scored hunter classes with multiple judges/rounds now route
   through the same per-judge rendering as derbies.

   Column maps confirmed:
   - Non-derby scored: R1 = col[15+j], R2 = col[24+j] (sequential per
     judge, up to 7 judges)
   - Derby: hiopt/base/bonus interleaved (completely different layout)

   Renderer auto-detects format from data shape:
   - phase.base exists → derby format (show base + hiopt + bonus)
   - phase.score exists → non-derby format (show score + phaseTotal)
   - renderPhaseMath(), renderPrecomputed(), renderPrecomputedByJudge()
     handle both formats
   - Bonus/hi-opt columns suppressed for non-derby scored classes
   - Per-judge ranks, judge card totals, movement arrows, split decision
     detection all work for non-derby scored classes
