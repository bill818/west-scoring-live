1. show.html "View Classes" unclickable when live (9ccbe44)
   The live pill was <a class="live-pill"> NESTED inside the
   <a class="ring-row"> CTA link. Nested anchors are invalid HTML —
   the browser auto-closes the outer .ring-row anchor before the
   pill, kicking "View Classes ▸" out of the link, so it could not
   be clicked whenever a class was live. Fix: render the pill as a
   <span role="link"> that navigates via JS (same .live-pill look,
   stopPropagation so it doesn't double-fire the row). Verified in
   preview: with live markup the CTA resolves back inside .ring-row,
   pill is a SPAN, no nested anchors. Bill follow-up confirmed the
   intent — BOTH paths must coexist: pill → live, rest of row →
   View Classes, plus the separate "Watch ▸" banner → live. All
   three verified independent.
