west-watcher.js — 1,482 lines — production ready with all features

WHAT IT DOES:
  - Polls C:\Ryegate\Jumper\Classes every 500ms
  - Reads config.dat for ring number and show slug
  - Reads config.json for Worker URL and auth key
  - Posts .cls changes to /postClassData
  - Posts UDP events to /postUdpEvent
  - Posts heartbeat to /heartbeat every 60s
  - Writes west_data.json locally on every change
  - Retry queue for failed POSTs (max 50)
  - Connectivity staleness warning after 5 min no successful post

UDP EVENT DETECTION:
  INTRO, CD_START, CD_STOPPED, CD_RESUMED,
  RIDE_START, FAULT, CLOCK_STOPPED, CLOCK_RESUMED,
  FINISH, UDP_SUSPENDED

UDP COLLISION DETECTION (NEW THIS SESSION):
  - Validates entry numbers against loaded .cls files
  - 3 consecutive mismatches → UDP suspended
  - 2 consecutive valid hits → auto-resumes
  - Unformatted (U) class → skip validation
  - No classes loaded → skip validation (startup safe)
  - Never gets stuck in waiting loop

UDP SILENCE DETECTION (NEW THIS SESSION):
  - Arms 3-second silence timer when ONCOURSE or CD
  - If no UDP packets for 3s → fires CLOCK_STOPPED or CD_STOPPED
    with reason: 'udp_silence'
  - Covers case where Ryegate stops packets on pause
    (vs frozen elapsed which existing timer covers)
  - Watch logs at HITS: reason field tells us which method Ryegate uses

ROUND INFERENCE:
  - Matches UDP TA to .cls header round config
  - Falls back to .cls entry state if ambiguous
  - ScoringMethod=9 two-phase → "Phase 1"/"Phase 2"
  - Labels: "Round 1", "Jump Off", "Round 3", "Phase 1", "Phase 2"

## THINGS TO REMEMBER (MISC)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

- Ryegate scraper (westscoring.live) is on shakier legal ground than watcher
  Pull it after HITS once new pipeline is verified solid
- .cls files and UDP are unencrypted open formats on hardware we operate
  Data is ours — horse names, times, results generated at our shows
- Ryegate col[26] is their back office ID — not reliable as show slug
  Setup wizard will let operator set slug manually
- 5-8 watchers can run simultaneously — all post to same Worker
  KV keys namespaced by slug:ring so they never collide
- $5/month Cloudflare paid plan needed before busy summer season
  Free tier KV write limit (1,000/day) will be hit with multiple active rings
- Horse/rider names must be stored consistently (same capitalization)
  This is the key that links results to follows later
- UDP pause behavior at HITS: watch for reason:'udp_silence' in logs
  If never seen → Ryegate sends frozen values (existing detector works)
  If seen → Ryegate stops packets (new silence detector works)
- Watcher is software-agnostic — when WEST software is ready
  just update file format parsing, pipeline stays identical
