PURPOSE
-------
This is the master punch list of EVERYTHING across our v3 planning
docs that is marked NOT CONFIRMED, STILL UNCERTAIN, SPEC ONLY, NEVER
OBSERVED LIVE, or flagged as a gap. When v3 testing begins — whether
at a specific show, via log replay, or via controlled fixtures — this
is the list to work through systematically.

Each item shows:
  [ ] or [X]           checkbox state
  Name + brief
  Source doc           where it's documented in detail
  What to confirm      what evidence closes the item
  How to close it      test approach (live, replay, fixture, spec-read)
  Priority             HIGH / MEDIUM / LOW

When an item is closed:
  1. Change [ ] to [X]
  2. Update the source doc with confirmation + date
  3. Remove the item from "STILL UNCERTAIN" sections in that doc
  4. Note the confirming class / log / session in a one-liner below

DO NOT close items by speculation. Close by evidence.
